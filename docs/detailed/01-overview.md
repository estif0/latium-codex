## 1. Introduction

Latium Codex (short: Latium) is a **Latin‑themed programming language** designed to be beginner‑friendly, educational, and automation‑focused. It transpiles to clean, readable Python code, making it easy to understand and debug while providing a rich set of built‑in features for file manipulation, system operations, and scripting.

The project is developed over a **6‑week core phase** with a subsequent extension for object‑oriented features. It aims to be a **professional‑grade language ecosystem** complete with a compiler toolchain, standard library, REPL, CLI, VS Code extension, and a documentation website.

## 3. Core Philosophy

- **Educational**: The language is designed with beginners in mind. Its syntax is clean and intuitive (indentation‑based, like Python), and it comes with comprehensive documentation, tutorials, and examples. Learning Latium should teach fundamental programming concepts that transfer to other languages.

- **Automation‑focused**: Latium includes built‑in file I/O, command execution, and a rich standard library. It aims to be a practical tool for automating everyday tasks, scripting, and prototyping.

- **Thematic**: All keywords, built‑in functions, and standard library modules are named in **Latin**, with a consistent grammatical style (verbs in imperative mood, nouns in nominative). Example programs and documentation are themed around Roman history and life, making the language memorable and distinctive.

- **Transpiler‑based**: Latium Codex does not compile to machine code or bytecode; instead, it **transpiles to Python**. This approach leverages Python’s vast ecosystem and runtime while keeping the generated code human‑readable. It also simplifies implementation and debugging.

- **Professional‑grade**: The project is built with software engineering best practices: modular architecture, unit tests, error reporting with line numbers and caret indicators, comprehensive documentation, and a full ecosystem (REPL, CLI, VS Code extension, website). It is designed to be maintainable, extensible, and suitable for real‑world use.

## 4. Target Audience

Latium Codex is intended for three main groups:

- **Students** learning programming concepts. The language’s simplicity, clear error messages, and educational materials make it an excellent first language.
- **Developers** seeking a lightweight scripting language with built‑in automation capabilities. Latium can be used for quick scripts, file processing, and system tasks.
- **Hobbyists** interested in language design, compiler construction, or Roman history. The project’s open‑source nature invites exploration and contribution.
